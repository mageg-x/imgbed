Telegram

⭐⭐⭐⭐⭐
利用 Telegram Bot 将图片发送到频道，获得永久 file_id，通过代理转换为可访问链接。
✓ 无限空间，单文件最大 50MB，稳定不删图
✗ 原始直链被墙，需配合代理

获取步骤

1
创建 Bot：打开 Telegram，搜索 @BotFather，发送 /newbot，按提示输入机器人名称和用户名（必须以 bot 结尾）。成功后你会收到 Bot Token（如 123456:ABCdefg），复制并保存。
2
创建频道：在 Telegram 中创建一个新的私有频道（任意名称），记下频道名。
3
邀请机器人：打开频道信息页 → 成员 → 添加成员，搜索你刚创建的机器人用户名，将其添加为管理员（至少需要"发送消息"和"上传文件"权限）。
4
获取频道 ID：在频道中随便发一条消息。然后在浏览器打开：https://api.telegram.org/bot你的BotToken/getUpdates，找到 "chat":{"id":-1001234567890，其中的 id 值（是一个负数）就是频道 ID。
5
填入 ImgBed：在 ImgBed 后台「存储管理」中选择 Telegram，填入 Bot Token 和 Channel ID，保存。
💡 注意：频道必须为私有，机器人必须具有发送消息和上传文件的权限。如需删除图片，直接在 Telegram 频道中删除对应消息即可。

Discord

⭐⭐⭐⭐
Discord 将上传的文件托管在 cdn.discordapp.com 全球 CDN 上，通过 Bot 发送消息到频道即可获得链接。
✓ 无限存储，单文件最大 25MB，自带 CDN
✗ 国内部分地区可能无法直接访问，需配合代理

获取步骤

1
创建应用：访问 Discord Developer Portal，点击「New Application」，输入名称（任意）。
2
创建 Bot：左侧菜单选择「Bot」，点击「Add Bot」→「Reset Token」并复制 Bot Token（类似 MTAzNzI...）。
3
设置权限：在 Bot 页面下方找到「Privileged Gateway Intents」，开启「Message Content Intent」。
4
邀请 Bot：左侧菜单选择「OAuth2」→「URL Generator」，勾选 bot 和 Send Messages、Attach Files、Read Messages、Manage Messages 权限，复制邀请链接并在浏览器中打开，选择你的服务器完成邀请。
5
获取频道 ID：在 Discord 客户端中打开「用户设置」→「高级」→ 开启「开发者模式」。然后右键点击你准备用于存储图片的文字频道，选择「复制频道 ID」。
💡 注意：Bot 必须拥有发送消息和上传文件的权限。建议为图片单独创建一个频道。Discord 的 CDN 链接有效期很长（理论永久），但若消息被删除则链接失效。
📖 Discord Developer Portal →

Hugging Face

⭐⭐⭐
Hugging Face 的 Space 支持存储任意文件，并通过 raw 链接公开访问，无需认证。
✓ 免费存储，技术社区常用，公开访问无需认证
✗ 存储空间有限，国内访问速度一般，需配合代理

获取步骤

1
注册账号：访问 Hugging Face 注册一个免费账号。
2
创建 Space：点击右上角头像 → 「New Space」。输入空间名称（如 my-imgbed），SDK 选择「Static」，Visibility 必须选择「Public」，点击「Create Space」。
3
生成 Access Token：点击右上角头像 → 「Settings」→ 「Access Tokens」→ 「New token」。名称任意，权限勾选「write」，点击「Generate token」，复制并保存（形如 hf_xxxxxx）。
4
填入 ImgBed：在 ImgBed 后台添加 Hugging Face 存储，填入 Access Token 和仓库名称（格式：用户名/空间名）。
💡 注意：Space 可以存储文件并公开访问，无需认证。原始直链地址为 https://huggingface.co/spaces/用户名/空间名/resolve/main/文件名，需配合代理使用。
📖 Hugging Face Spaces 文档 →

Cloudflare R2

⭐⭐⭐⭐
Cloudflare R2 提供 S3 兼容的对象存储，免费额度足够个人图床。
✓ 10GB 免费存储 + 每月 10GB 出站流量，全球 CDN 加速
✗ 需要绑定信用卡（但不会扣费，仅验证身份）

获取步骤

1
注册 Cloudflare：访问 Cloudflare 注册账号。
2
开通 R2：登录后，左侧菜单选择「R2」→ 点击「开始使用」→ 根据提示添加付款方式（支持信用卡，不会扣费，仅用于防滥用）。
3
创建存储桶：点击「创建存储桶」，输入名称（如 imgbed），区域选择「自动」，点击创建。
4
生成 API 令牌：进入存储桶详情页，点击「设置」→「API 令牌」→「创建 API 令牌」。权限选择「对象读写」，创建后会显示 Access Key ID 和 Secret Access Key，务必立即复制保存。
5
填入 ImgBed：在 ImgBed 后台选择「S3 兼容存储」，填入端点（https://你的账户ID.r2.cloudflarestorage.com）、区域（auto）、存储桶名称、访问密钥和秘密密钥。
💡 注意：免费额度每月重置，超出部分会收费，建议在 Cloudflare 设置中启用「限制 R2 使用量」。通过 ImgBed 代理访问时，流量走代理，不计入 R2 出站流量，可节省额度。

网盘挂载

⭐⭐⭐
使用 Alist 将网盘挂载为 WebDAV 或 S3 接口，ImgBed 通过 WebDAV 连接，实现超大容量存储。
✓ 网盘容量大（通常 2TB+），单文件可达 100MB+
✗ 需要额外维护 Alist 服务

获取步骤

1
部署 Alist：下载 Alist 二进制文件运行，或使用 Docker：docker run -d -p 5244:5244 xhofe/alist:latest
2
添加网盘存储：打开 Alist 管理后台（默认 http://localhost:5244），进入「存储」→「添加」，选择驱动类型（如「阿里云盘」），根据提示获取 Refresh Token。
3
开启 WebDAV：Alist 默认开启 WebDAV 服务，地址为 http://localhost:5244/dav。用户名和密码与你登录 Alist 后台的相同。
4
填入 ImgBed：在 ImgBed 后台选择「WebDAV」，填入端点、用户名、密码和路径前缀。
💡 注意：如果 Alist 部署在本地，ImgBed 需能访问该地址。网盘直链有效期由网盘决定，Alist 会在访问时自动刷新。建议选择支持 Alist 官方文档中标记为「稳定性高」的网盘（如阿里云盘、夸克网盘）。

Cloudflare Workers

推荐

全球边缘网络，每日 10 万次请求免费，绑定自定义域名后国内访问速度尚可。

部署步骤

1
点击下方按钮，登录你的 Cloudflare 账号（如果没有，免费注册一个）。
2
按页面提示为 Worker 命名（例如 imgbed-proxy），点击「部署」。
3
部署成功后，你会获得一个形如 https://你的项目名.workers.dev 的域名。复制该域名。
4
回到 ImgBed 后台的「代理管理」，粘贴该地址，保存。

❓ 常见问题

FAQ

我部署的 Cloudflare Worker 代理访问时返回 404？
检查 ImgBed 中填写的代理地址是否正确（不要遗漏 https://），并确认代理已启用。另外，ImgBed 生成的链接格式为 https://代理域名/base58(原始host)/图片路径，如果手动修改过代理代码导致路由变化，请重新部署。
Telegram 频道 ID 是负数，ImgBed 支持吗？
支持，直接粘贴即可。
Discord 的 Bot Token 重置后，之前的图片还能访问吗？
可以，只要频道和消息还在，图片链接依然有效。但新上传需要使用新 Token。
Hugging Face 上传后，直接访问原始链接被限制？
请确认你的仓库是 Public 状态。另外 Hugging Face 有时会要求登录才能下载，建议始终通过代理访问。
网盘挂载方式会不会很慢？
Alist 会在第一次访问时获取网盘直链，之后 ImgBed 代理会缓存，速度取决于网盘和代理。推荐使用阿里云盘，速度较快。
代理域名国内访问还是慢怎么办？
可以尝试绑定自己的域名并开启 Cloudflare 代理，或使用又拍云等国内 CDN 加速（见上文「白嫖 CDN」部分）。
ImgBed 本身收费吗？
完全免费开源，永不收费。项目地址：GitHub - mageg-x/imgbed
