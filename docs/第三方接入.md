# ImgBed 第三方接入文档

## 一、产品定位说明

ImgBed 是一个**免费图床聚合工具**，专注于：

- ✅ **代理上传流量**：图片上传经过 ImgBed 分发到各个免费存储渠道
- ❌ **不代理访问流量**：图片直接使用原始存储渠道的 URL，节省服务器带宽

---

## 二、认证方式设计

### 2.1 两种认证模式

| 认证方式 | 说明 | 使用场景 |
|---------|------|---------|
| **匿名上传** | 无需认证，直接上传 | 快速测试、临时使用 |
| **Token 认证** | 使用 API Token 认证 | 博客编辑器、脚本集成、长期使用 |

---

## 三、匿名上传接口

### 3.1 接口说明

**无需任何认证**，可直接调用，但有以下限制：

- 限速：每分钟最多 5 次上传
- 文件大小限制：最大 5MB
- 不支持批量上传
- 不记录上传历史（无法管理）

### 3.2 接口详情

**上传图片：**
```http
POST /api/v1/upload/anonymous
Content-Type: multipart/form-data

Form Data:
- file: 图片文件 (必填)
```

**响应示例：**
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "id": "abc123",
    "name": "image.jpg",
    "url": "https://cdn.discord.com/attachments/xxx/image.jpg",
    "size": 102400,
    "type": "image/jpeg",
    "channel": "discord",
    "links": {
      "url": "https://cdn.discord.com/attachments/xxx/image.jpg",
      "markdown": "![image](https://cdn.discord.com/attachments/xxx/image.jpg)",
      "html": "<img src=\"https://cdn.discord.com/attachments/xxx/image.jpg\" alt=\"image\">"
    }
  }
}
```

### 3.3 使用示例

**curl：**
```bash
curl -X POST https://your-imgbed.com/api/v1/upload/anonymous \
  -F "file=@/path/to/image.jpg"
```

**JavaScript (浏览器)：**
```javascript
async function uploadAnonymous(file) {
    const formData = new FormData();
    formData.append('file', file);
    
    const response = await fetch('https://your-imgbed.com/api/v1/upload/anonymous', {
        method: 'POST',
        body: formData
    });
    
    return await response.json();
}
```

## 四、Token 认证接口（推荐）

### 4.1 Token 创建流程

1. 登录 ImgBed 管理后台
2. 进入 **Tokens** 页面
3. 点击 **创建 Token**
4. 填写信息：
   - Token 名称（如："我的博客"）
   - 权限（如：仅 `upload`）
   - 过期时间（可选）
5. 保存好生成的 `token` 和 `secret`（只显示一次！）

### 4.2 Token 权限说明

| 权限 | 说明 |
|------|------|
| `upload` | 上传图片 |
| `upload:multiple` | 批量上传 |
| `read` | 读取文件列表和信息 |
| `delete` | 删除文件 |
| `*` | 所有权限 |

### 4.3 认证方式

**在请求 Header 中携带：**
```http
X-API-Token: your_token_here
X-API-Secret: your_secret_here
```

### 4.4 Token 认证接口列表

| 接口 | 方法 | 所需权限 | 说明 |
|------|------|---------|------|
| `/api/v1/upload` | POST | `upload` | 上传单张图片 |
| `/api/v1/upload/multiple` | POST | `upload:multiple` | 批量上传图片 |
| `/api/v1/files` | GET | `read` | 获取上传的文件列表 |
| `/api/v1/file/:id` | DELETE | `delete` | 删除指定文件 |
| `/api/v1/files` | DELETE | `delete` | 批量删除文件 |

## 五、上传接口详解

### 5.1 单张上传（Token 认证）

**接口：**
```http
POST /api/v1/upload
Content-Type: multipart/form-data
X-API-Token: your_token
X-API-Secret: your_secret

Form Data:
- file: 图片文件 (必填)
- directory: 目录路径 (可选)
- tags: 标签数组 (可选)
```

**响应：**
```json
{
  "code": 0,
  "message": "success",
  "data": {
    "id": "abc123",
    "name": "image.jpg",
    "originalName": "my-photo.jpg",
    "url": "https://cdn.discord.com/attachments/xxx/image.jpg",
    "size": 2048000,
    "type": "image/jpeg",
    "channel": "discord",
    "channelType": "discord",
    "uploadedAt": 1712345678,
    "links": {
      "url": "https://cdn.discord.com/attachments/xxx/image.jpg",

      "bbcode": "[img]https://cdn.discord.com/attachments/xxx/image.jpg[/img]"

```

**接口：**
```http
POST /api/v1/upload/multiple
Content-Type: multipart/form-data
X-API-Token: your_token
X-API-Secret: your_secret

```

**响应：**

## 六、使用场景示例

### 6.1 场景 1：Typora 配置

在 Typora 的「偏好设置」→「图像」中：

- **上传服务**：选择 `Custom Command`
- **命令**：
```bash
curl -X POST https://your-imgbed.com/api/v1/upload/anonymous \
  -F "file=@${filepath}" \
  | grep -o '"url":"[^"]*"' | cut -d'"' -f4
```

**或者使用 Token（推荐）：**
```bash
curl -X POST https://your-imgbed.com/api/v1/upload \
  -H "X-API-Token: your_token" \
  -H "X-API-Secret: your_secret" \
  -F "file=@${filepath}" \
  | grep -o '"url":"[^"]*"' | cut -d'"' -f4
```

### 6.2 场景 2：VS Code 插件配置

在插件配置中添加：

```json
{
  "imgbed.uploadUrl": "https://your-imgbed.com/api/v1/upload",
  "imgbed.token": "your_token",
  "imgbed.secret": "your_secret"
}
```

### 6.3 场景 3：Python 脚本上传

```python
import requests

class ImgBedClient:
    def __init__(self, base_url, api_token=None, api_secret=None):
        self.base_url = base_url
        self.api_token = api_token
        self.api_secret = api_secret
    
    def upload(self, image_path):
        with open(image_path, 'rb') as f:
            files = {'file': f}
            
            if self.api_token and self.api_secret:
                headers = {
                    'X-API-Token': self.api_token,
                    'X-API-Secret': self.api_secret
                }
                url = f"{self.base_url}/api/v1/upload"
            else:
                headers = {}
                url = f"{self.base_url}/api/v1/upload/anonymous"
            
            response = requests.post(url, files=files, headers=headers)
            return response.json()

# 使用 Token（推荐）
client = ImgBedClient(
    "https://your-imgbed.com",
    api_token="your_token",
    api_secret="your_secret"
)

# 匿名上传
# client = ImgBedClient("https://your-imgbed.com")

result = client.upload("image.jpg")
if result['code'] == 0:
    print(f"上传成功: {result['data']['links']['markdown']}")
```

### 6.4 场景 4：JavaScript 博客编辑器集成

```javascript
class ImgBedUploader {
    constructor(options) {
        this.baseUrl = options.baseUrl;
        this.apiToken = options.apiToken;
        this.apiSecret = options.apiSecret;
    }
    
    async upload(file) {
        const formData = new FormData();
        formData.append('file', file);
        
        const headers = {};
        let url;
        
        if (this.apiToken && this.apiSecret) {
            headers['X-API-Token'] = this.apiToken;
            headers['X-API-Secret'] = this.apiSecret;
            url = `${this.baseUrl}/api/v1/upload`;
        } else {
            url = `${this.baseUrl}/api/v1/upload/anonymous`;
        }
        
        const response = await fetch(url, {
            method: 'POST',
            headers,
            body: formData
        });
        
        return await response.json();
    }
}

// 在编辑器中使用
const uploader = new ImgBedUploader({
    baseUrl: 'https://your-imgbed.com',
    apiToken: 'your_token',
    apiSecret: 'your_secret'
});

// 监听文件选择
document.getElementById('file-input').addEventListener('change', async (e) => {
    const file = e.target.files[0];
    const result = await uploader.upload(file);
    
    if (result.code === 0) {
        // 插入 Markdown 链接到编辑器
        insertMarkdown(result.data.links.markdown);
    }
});

// 监听粘贴事件（支持直接粘贴图片）
document.addEventListener('paste', async (e) => {
    const items = (e.clipboardData || e.originalEvent.clipboardData).items;
    
    for (const item of items) {
        if (item.kind === 'file' && item.type.startsWith('image/')) {
            e.preventDefault();
            
            const file = item.getAsFile();
            const result = await uploader.upload(file);
            
            if (result.code === 0) {
                insertMarkdown(result.data.links.markdown);
            }
            break;
        }
    }
});
```

### 6.5 场景 5：仅粘贴上传（最简单）

如果你只需要支持复制粘贴图片，可以直接用这个简化版本：

```html
<!DOCTYPE html>
<html>
<body>
    <textarea id="editor" placeholder="在这里写文章，Ctrl+V 粘贴图片..."></textarea>
    
    <script>
        const BASE_URL = 'https://your-imgbed.com';
        const API_TOKEN = 'your_token';   // 可选
        const API_SECRET = 'your_secret'; // 可选
        
        async function uploadImage(file) {
            const formData = new FormData();
            formData.append('file', file);
            
            const headers = {};
            let url;
            
            if (API_TOKEN && API_SECRET) {
                headers['X-API-Token'] = API_TOKEN;
                headers['X-API-Secret'] = API_SECRET;
                url = `${BASE_URL}/api/v1/upload`;
            } else {
                url = `${BASE_URL}/api/v1/upload/anonymous`;
            }
            
            const response = await fetch(url, {
                method: 'POST',
                headers,
                body: formData
            });
            
            return await response.json();
        }
        
        // 监听粘贴事件
        document.getElementById('editor').addEventListener('paste', async (e) => {
            const items = e.clipboardData.items;
            
            for (const item of items) {
                if (item.kind === 'file' && item.type.startsWith('image/')) {
                    e.preventDefault();
                    
                    const file = item.getAsFile();
                    const result = await uploadImage(file);
                    
                    if (result.code === 0) {
                        const markdown = result.data.links.markdown;
                        const textarea = e.target;
                        const start = textarea.selectionStart;
                        const end = textarea.selectionEnd;
                        const text = textarea.value;
                        
                        textarea.value = text.substring(0, start) + markdown + text.substring(end);
                        textarea.selectionStart = textarea.selectionEnd = start + markdown.length;
                    }
                    break;
                }
            }
        });
    </script>
</body>
</html>
```

## 七、匿名 vs Token 对比

| 特性 | 匿名上传 | Token 认证 |
|------|---------|-----------|
| 认证 | 无需认证 | 需要 Token + Secret |
| 限速 | 5次/分钟 | 无限制（受平台限制） |
| 文件大小 | 最大 5MB | 最大 20MB（受渠道限制） |
| 批量上传 | ❌ 不支持 | ✅ 支持 |
| 文件管理 | ❌ 无法管理 | ✅ 可查看、删除 |
| 使用历史 | ❌ 不记录 | ✅ 完整记录 |
| 适用场景 | 临时测试、快速上传 | 博客、编辑器、长期使用 |

---

## 八、安全建议

1. **优先使用 Token**：匿名上传仅用于测试，正式使用请创建 Token
2. **权限最小化**：只给必要的权限（如仅 `upload`）
3. **定期轮换 Token**：建议每 3-6 个月更换一次
4. **使用 HTTPS**：确保 Token 传输安全
5. **不要泄露 Token**：不要将 Token 提交到公开代码仓库
