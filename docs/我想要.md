# 「我想要」页面设计需求文档（修订版）

## 1. 项目背景与目标

### 1.1 产品实质
- **ImgBed** 是一个**自建图床管理工具**，以单二进制形式分发。
- **核心能力**：用户配置自己的存储渠道（Telegram Bot、Discord Webhook、GitHub Token 等）和代理地址（Cloudflare Worker、Vercel Edge 等）后，ImgBed 提供统一的上传界面和 REST API，自动将上传的图片存入用户指定的免费存储，并将原始直链转换为用户配置的代理/CDN 地址返回。
- **用户需要自行白嫖**：Telegram Bot Token、Discord 频道、GitHub Token、Cloudflare Worker 脚本等。ImgBed 不提供这些资源，但提供**配置向导**和**一键生成代理模板**。

### 1.2 页面目标
- 吸引访客下载并使用 ImgBed。
- **诚实传达**：用户需要自行获取免费存储和代理，但过程简单，程序会辅助。
- 提供清晰的**获取资源链接**和**配置示例**，降低用户的心理门槛。

## 2. 页面定位与入口

- 页面名称：`get-started.html` 或 `i-want.html`
- 入口：官网导航栏高亮按钮「**我想要**」
- 页面基调：**实用、直白、行动导向**，略带“薅羊毛”的兴奋感。

## 3. 页面结构

### 3.1 Hero 区域：一句话说明白
- **主标题**：`「我想要」一个属于自己的免费图床`
- **副标题**：`下载 ImgBed，填入你白嫖的 Token，立刻拥有无限存储 + 国内加速`
- **核心提示**（醒目卡片）：`你不需要写代码，只需要准备：Telegram Bot / Discord / GitHub 任选其一 + Cloudflare Worker 代理地址`

### 3.2 三步上手指南（重点）

#### 第一步：获取免费存储
- 用图标+短说明展示三种主流白嫖渠道：
  - **Telegram**：创建 Bot → 获得 Token，创建频道 → 获得 Channel ID。[?] 帮助链接
  - **Discord**：创建应用 → 获得 Bot Token，创建频道 → 获得频道 ID。[?] 帮助链接
  - **GitHub**：创建公开仓库 → 生成 Personal Access Token。[?] 帮助链接
- 每个渠道旁提供**官方教程链接**（新窗口打开）。
- 提示：`全部免费，终身有效`

#### 第二步：获取免费代理/CDN
- 说明：`你的图片直链在国内可能无法访问，你需要一个代理来加速。`
- 推荐方案：
  - **Cloudflare Workers**：提供一键复制 Worker 代码的按钮，用户只需粘贴部署，获得 `xxx.workers.dev` 地址。
  - **Vercel Edge Functions**：提供一键部署按钮或代码模板。
- 提供**图文教程链接**（可折叠或弹窗）。

#### 第三步：运行 ImgBed 并配置
- 下载适合的二进制文件。
- 双击运行 → 浏览器打开 `http://localhost:8380`。
- 在 Web 管理后台填入你刚才获取的 Token 和代理地址。
- 开始上传，自动获得代理后的图片链接。

### 3.5 常见顾虑打消（FAQ 折叠块）
- **Q：需要服务器吗？**  
  A：不需要。桌面版直接运行，服务端可部署在免费云（如 Oracle Cloud 永久免费 VPS）。
- **Q：图片会被删除吗？**  
  A：存储在你自己的 Telegram/Discord/GitHub 账号下，只要你的账号不被封，图片永存。
- **Q：代理地址会不会失效？**  
  A：代理由你自己部署在 Cloudflare/Vercel 免费套餐上，你拥有完全控制权。
- **Q：我能用自己的域名吗？**  
  A：可以，绑定自定义域名到你的 Worker 即可，程序支持配置。
